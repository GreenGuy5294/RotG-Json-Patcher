Tabletop Simulator JSON Patcher

About

This is a tool for updating Tabletop Simulator .json files into the cached version of Rise of the Guys. If you made your deck before v1.0.3, the links need to be changed. Simply copy and paste the text of a .json file into the textarea above and hit the patch button to update all image links!

Windows: By default, your .json files will be located inside "C:\Users\[username]\Documents\My Games\Tabletop Simulator\Saves", and saved objects are inside the Saved Objects folder.

Why does this tool exist?

Tabletop Simulator caches images by default to help load things faster. I was using Imgur to host the images, since I believed it was not possible to bypass the cache on Tabletop Simulator. I now host my images on Github and can tell Tabletop Simulator to verify the cache each time the game is opened.

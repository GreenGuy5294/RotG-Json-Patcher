About

This is a tool for updating Tabletop Simulator .json files into the latest version of Rise of the Guys. Whenever the game updates, the images need to be changed. Simply copy and paste the text of a .json file into the textarea above and hit the patch button to update all image links!

Windows: By default, your .json files will be located inside "C:\Users\[username]\Documents\My Games\Tabletop Simulator\Saves"

Why does this tool exist?

Tabletop Simulator caches images by default to help load things faster. It recognizes them by link, so if it's a link that it's downloaded from before it will pull an old version from the game files rather than download it again, even if the image is updated by the link. Because of this, I use Imgur to host images, which doesn't allow for images to update. I created this tool to easily update the decks.
